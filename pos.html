POS Machine

POS is the Point of Sale. It corresponds to a location where a
transaction occurs when purchasing or billing for a service or product,
such as a market, supermarket, and so on.

POS(Point Of Sale) Machine system

POS Machine processes the transaction between a merchant and the customer.

A POS machine is actually an electronic device which is used to process card payments.

Common features that every POS machine has are credit and debit card acceptance.
